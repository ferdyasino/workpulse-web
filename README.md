# WorkPulse

A modern Attendance & Payroll Management System built with React, TypeScript, and Vite.

---

## Features

- Attendance Management
- Employee Management
- Department Management
- Position Management
- Shift Scheduling
- Payroll
- Reports
- Google Authentication
- Role-based Access Control
- Responsive Dashboard

---

## Tech Stack

### Frontend

- React 19
- TypeScript
- Vite 8
- Material UI
- React Router
- Axios

### Backend (Current)

- Google Apps Script
- Google Sheets

### Backend (Future)

- Node.js
- PostgreSQL

---

## Development

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
